# Installation

You must install the CLI globally.

```bash title="Terminal"
npm install -g rehooks-cli
```

# Usage

```bash title="Terminal"
rehooks
```

## Options

CLI options are as follows:

| Option            | Description                 |
| ----------------- | --------------------------- |
| `-v`, `--version` | Display the version number. |
| `-h`, `--help`    | Display help for command.   |

## Commands

CLI commands are as follows:

| Command          | Description                          |
| ---------------- | ------------------------------------ |
| `hooks <file>`   | Import hooks to your component file. |
| `help [command]` | Display help for command.            |

## Command Usage

```bash title="Terminal"
rehooks hooks ./path/to/component
```

this command will import hooks to your component by giving the path to the component file.
