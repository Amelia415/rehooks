# Installation

You must install the CLI globally.

```bash title="Terminal"
npm install -g rehooks-cli
```

or you can execute the command with npx.

```bash title="Terminal"
npx rehooks-cli
```

# Usage

```bash title="Terminal"
rehooks
```

## Options

These are the options available in the CLI.

| Option          | Description                 |
| --------------- | --------------------------- |
| `-v, --version` | Displays the version number |
| `-h, --help`    | Displays help for command   |

## Commands

| Command          | Description                     |
| ---------------- | ------------------------------- |
| `hooks <file>`   | Import hooks to your component. |
| `help <command>` | Displays help for command       |

## Example

This is how you execute a command to import hooks to your component.

```bash title="Terminal"
npx rehooks-cli hooks ./src/components/component.tsx
```

> The result will be something like this.

```bash title="Terminal"
✔ Select hooks to import: useCounter, useFocus, useScroll

  ○ useCountDown
  ○ useCounter
  ○ useDebounceCallback
  ○ useDebounceValue
  ○ useEventCallback
  ○ useEventListener

  ● useScroll
```
