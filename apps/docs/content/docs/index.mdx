# What is Rehooks?

Rehooks is an open-source library that provides a wide range of react reusable hooks. The library is written in TypeScript and bundled with `tsup` for maximum performance and compatibility.

# Why use Rehooks?

Rehooks is a collection of reusable hooks that can be used to simplify your codebase and improve performance. It provides a wide range of hooks for common use cases, such as form validation, data fetching, and more. By using Rehooks, you can reduce the amount of code you need to write and focus on the core functionality of your application.

# Key Features

- 🔥 Lightweight and performant: Rehooks is written in TypeScript and bundled with `tsup` for maximum performance and compatibility.
- 📦 Modular and tree-shakable: Rehooks is designed to be modular and tree-shakable, allowing you to only include the hooks you need in your application.
- 📝 Well-documented: Rehooks comes with detailed documentation and examples, making it easy to understand and use.
- 💪 Tested and maintained: Rehooks has a comprehensive test suite and is actively maintained by the community.
